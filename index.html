Módulo 1: Fundamentos Web (HTML, CSS, JavaScript)

Aprende los pilares del desarrollo web frontend con una guía completa de HTML5, CSS3 y JavaScript moderno.

1

Semántica HTML5

¿Qué es la semántica en HTML5?

La semántica en HTML5 se refiere al uso de etiquetas que describen el significado del contenido en lugar de solo su apariencia. Esto mejora la accesibilidad, el SEO y la mantenibilidad del código.

Consejo: Usa elementos semánticos para estructurar tu contenido. Evita el uso excesivo de divs cuando exista una etiqueta más descriptiva.
HTML
<header>
  <h1>Título del Sitio</h1>
  <nav>
    <a href="#">Inicio</a>
    <a href="#">Blog</a>
  </nav>
</header>

<main>
  <article>
    <h2>Título del Artículo</h2>
    <p>Contenido principal...</p>
  </article>
  
  <aside>
    <h3>Contenido Relacionado</h3>
    <p>Información adicional...</p>
  </aside>
</main>

<footer>
  <p>© 2023 Mi Sitio Web</p>
</footer>

Elementos Semánticos Clave:

  • <header>: Cabecera de la página o sección
  • <nav>: Contiene los enlaces de navegación
  • <main>: Contenido principal único en la página
  • <article>: Contenido independiente y autocontenido
  • <section>: Agrupación temática de contenido
  • <aside>: Contenido relacionado indirectamente
  • <footer>: Pie de página o sección
  • <figure> y <figcaption>: Para imágenes y su descripción

Flexbox y CSS Grid

Sistemas de Layout Modernos

Flexbox y CSS Grid son dos poderosos sistemas de diseño en CSS que permiten crear layouts complejos y responsivos con menos código.

Regla práctica: Usa Flexbox para diseños unidimensionales (en una dirección) y CSS Grid para diseños bidimensionales (filas y columnas).
CSS
/* Flexbox Example */
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

/* CSS Grid Example */
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  grid-gap: 20px;
}

/* Grid Area Example */
.header { grid-area: header; }
.main { grid-area: main; }
.sidebar { grid-area: sidebar; }

.app-layout {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main";
  grid-template-columns: 1fr 3fr;
}

Módulo 2: Desarrollo con React.js

Aprende a construir aplicaciones modernas con React, el framework de JavaScript más popular.

1

Componentes y Props

Fundamentos de React

React se basa en componentes reutilizables que reciben datos a través de props (propiedades) y mantienen su propio estado interno.

JSX
// Componente funcional
function Welcome(props) {
  return <h1>Hola, {props.name}</h1>;
}

// Componente de clase
class Welcome extends React.Component {
  render() {
    return <h1>Hola, {this.props.name}</h1>;
  }
}

// Uso del componente
function App() {
  return (
    <div>
      <Welcome name="Sara" />
      <Welcome name="Carlos" />
    </div>
  );
}
2

Hooks Avanzados

Gestión de Estado y Efectos

Los Hooks son funciones que te permiten "enganchar" el estado de React y el ciclo de vida desde componentes funcionales.

JSX
import { useState, useEffect, useContext, useReducer } from 'react';

// useState: Gestión de estado local
function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Contador: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Incrementar
      </button>
    </div>
  );
}

// useEffect: Efectos secundarios
function DataFetcher({ userId }) {
  const [data, setData] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      const response = await fetch(`https://api.example.com/users/${userId}`);
      const jsonData = await response.json();
      setData(jsonData);
    };

    fetchData();
    
    // Cleanup function
    return () => {
      // Cancelar peticiones o limpiar suscripciones
    };
  }, [userId]); // Dependencias

  return <div>{data ? data.name : 'Cargando...'}</div>;
}

Módulo 3: Backend con Node.js y MySQL

Aprende a construir servidores robustos con Node.js y a interactuar con bases de datos MySQL.

1

Express.js: Creación de APIs

Framework Express para Node.js

Express es el framework más popular para Node.js que simplifica la creación de APIs RESTful y aplicaciones web.

JavaScript
const express = require('express');
const app = express();
const port = 3000;

// Middleware para parsear JSON
app.use(express.json());

// Ruta GET básica
app.get('/', (req, res) => {
  res.send('¡Hola Mundo!');
});

// Ruta con parámetros
app.get('/users/:id', (req, res) => {
  const userId = req.params.id;
  // Lógica para obtener usuario...
  res.json({ id: userId, name: "Juan Pérez" });
});

// Ruta POST para crear recursos
app.post('/users', (req, res) => {
  const newUser = req.body;
  // Lógica para guardar el usuario...
  res.status(201).json(newUser);
});

// Manejo de errores
app.use((err, req, res, next) => {
  console.error(err.stack);
  res.status(500).send('¡Algo salió mal!');
});

app.listen(port, () => {
  console.log(`Servidor escuchando en http://localhost:${port}`);
});
2

Conexión a MySQL

Interactuando con Bases de Datos

MySQL es un sistema de gestión de bases de datos relacionales ampliamente utilizado. Conecta tu aplicación Node.js a MySQL para almacenar y recuperar datos.

JavaScript
const mysql = require('mysql2');

// Crear conexión a la base de datos
const pool = mysql.createPool({
  host: 'localhost',
  user: 'root',
  password: 'password',
  database: 'my_database',
  waitForConnections: true,
  connectionLimit: 10,
  queueLimit: 0
});

// Promisify para usar async/await
const promisePool = pool.promise();

// Ejemplo de consulta
const getUsers = async () => {
  try {
    const [rows] = await promisePool.query('SELECT * FROM users');
    return rows;
  } catch (error) {
    console.error('Error al obtener usuarios:', error);
    throw error;
  }
};

// Ejemplo de inserción
const createUser = async (userData) => {
  const { name, email } = userData;
  try {
    const [result] = await promisePool.query(
      'INSERT INTO users (name, email) VALUES (?, ?)',
      [name, email]
    );
    return { id: result.insertId, ...userData };
  } catch (error) {
    console.error('Error al crear usuario:', error);
    throw error;
  }
};
